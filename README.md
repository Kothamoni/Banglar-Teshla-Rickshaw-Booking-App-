Installation Steps
1. Clone the Repository
bash
git clone https://github.com/yourusername/rickshaw-booking-system.git
cd rickshaw-booking-system
2. Frontend Setup
bash
# Install dependencies (if using package.json)
npm install

# Or manually create structure
mkdir -p public/{css,js,images,video}
3. Backend Setup (Node.js/Express)
bash
# Create backend directory
mkdir backend
cd backend

# Initialize Node.js project
npm init -y

# Install required packages
npm install express mongoose cors jsonwebtoken bcryptjs dotenv
npm install -D nodemon

# Create environment file
echo "PORT=3000
MONGODB_URI=mongodb://localhost:27017/rickshaw_db
JWT_SECRET=your_super_secret_jwt_key_here
PAYMENT_API_KEY=your_sslcommerz_api_key" > .env
4. Database Setup
bash
# Start MongoDB service
mongod

# In another terminal, connect to MongoDB
mongo

# Create database
use rickshaw_db

# Create collections (via application)
5. Running the Application
bash
# Terminal 1: Start backend server
cd backend
npm run dev

# Terminal 2: Serve frontend files
# Using Python's simple HTTP server
python -m http.server 5500

# Or using Node.js serve
npx serve .
6. Access the Application
Frontend: http://localhost:5500

Backend API: http://localhost:3000

Database: mongodb://localhost:27017
